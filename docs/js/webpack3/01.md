# 01-为什么需要webpack

过去的代码和未来的代码的进行改变,以前前端只有View层,现在是前后端分离,因此出现了渲染层和数据层,路由层,视图层,对应的后端只有api和数据层

## 为什么需要webpack

![为什么需要webpack.png][1]

过去的时候有,很多库,不过这些库都是一些`dom`的方法
`dojo`,`yui`,`jquery`,`prototype`,`o`,`kissy`,基本上都是操作`dom`的方法封装
也就是一直说的`MVC`时代,

通常是`Backbone.js`和`underscore.js`,`jquery`为代表
