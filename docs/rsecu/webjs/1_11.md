# babel

网上关于babel的概念很多，这里把概念抽离，Babel是一个平台，他有很多插件，babel这个平台可以让多个插件良好的使用。

他是把一堆程序员使用特别新型的JS语法，通过一些算法，转成大多的老版本浏览器可以识别的代码。

一般的，几乎所有的程序都会有babel的引入。
